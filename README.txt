
# 🔐 Password Cracking and Protection Toolkit

Welcome to the **Password Cracking and Protection Toolkit**! This project is designed to help you explore password hashing, encryption, and validation techniques, while offering a clean and interactive interface.

---

## 🚀 **Project Overview**

This application includes:
1. **Password Features**:
   - **Password Hashing** (SHA-256)
   - **Password Encryption** (AES-based encryption)
   - **Password Policy Validation**
2. **User-Friendly Interface**: A clean and intuitive GUI built with Tkinter.
3. **Secure Key Management**: Generate and store encryption keys securely.

---

## 🛠️ **Technologies Used**

- **Python 3**: Core programming language.
- **Tkinter**: GUI interface.
- **Cryptography**: AES encryption for passwords.
- **Hashlib**: SHA-256 hashing.

---

## 💻 **Installation**

Follow these steps to set up the project:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-repository/password_toolkit.git
   cd password_toolkit
   ```

2. **Install Dependencies**:
   Run the following command to install required libraries:
   ```bash
   pip3 install cryptography
   sudo apt install python3-tk  # For Tkinter (Linux only)
   ```

3. **Run the Application**:
   ```bash
   python3 main.py
   ```

---

## 📋 **Features**

### 1. **Password Hashing**
- Hash any password securely using SHA-256.

### 2. **Password Encryption**
- Encrypt passwords securely using AES encryption.
- Generate and display encryption keys.

### 3. **Password Policy Check**
- Validate passwords against security policies (length, uppercase, numbers, etc.).

### 4. **Interactive GUI**
- A clean and user-friendly interface built using Tkinter.
